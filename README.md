# Kids Games

A collection of educational games for children.

## Games

### Multiplication Game
A tablet-optimized multiplication learning game for kids aged 7-9.

**Location:** `multiplication/multiplication-game.html`

**Features:**
- Practice multiplication tables (2-10)
- Multiple game modes: Result mode and Multiplier mode
- Fixed session or Practice All combinations
- Learn mode to view multiplication tables
- Tablet-friendly touch interface
